JavaScript

Javascript é a linguagem de script mais popular da web, e funciona bem na maioria dos principais navegadores (browsers), como Internet Explorer, Firefox, Chrome, Safari e Opera. Pré-requisitos para aprender Javascript: Antes de aprender Javascript, você deve conhecer pelo menos o básico dos seguintes assuntos: HTML/XHTML

O que é Javascript?
Informação

Bem vindo ao mundo JS!

JavaScript foi desenvolvido para adicionar interatividade às páginas HTML. Javascript é uma linguagem de scripting. Uma linguagem de script é uma linguagem de programação leve. JavaScript é normalmente embutido diretamente em páginas HTML. JavaScript é uma linguagem interpretada (significa que scripts executam sem compilação preliminar). Todos podem usar o JavaScript sem a aquisição de uma licença.

O que o Javascript pode fazer?
Javascript fornece aos webdesigners uma linguagem de programação. Com sintaxe simples e sendo uma linguagem leve, mesmo webdesigners, que não são programadores, podem adicionar funcionalidades dinâmicas nas suas páginas web. Javascript pode colocar textos dinâmicos em páginas web. Textos que mudam com frequência e automáticamente, como o nome do usuário, bom dia, boa tarde, boa noite, data e horário, podem ser implementados em páginas web com Javascript. Javascript pode reagir a eventos.
